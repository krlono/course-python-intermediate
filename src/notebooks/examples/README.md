# Examples – Data Processing in Python

This folder contains short and practical examples of common concepts in data processing using Python and pandas.

***

## Contents

### 📁 Exercises

* ./0\_excercises.py

***

### 📁 Basic Concepts

* ./aggregation.py  
  Aggregation of data (e.g. `groupby`, summing, counting).

* ./data\_types.py  
  Overview of basic data types in Python.

* ./functs.py  
  Introduction to functions: what they are and when to use them.

* ./merge\_files.py  
  Joining datasets, e.g. inner join, left join, cross join, anti-join.

***

### 📁 File Handling

* ./glob\_file\_search.py  
  How to find files using `glob`, including wildcards and recursive searches.

***

### 📁 Date and Time

* ./date\_time\_handling.py  
  How to work with dates in pandas, including the use of `.dt`.

***

### 📁 Lists and Data Structures

* ./lists\_sets\_tuples.py  
  The difference between Python data types such as list, set, and tuple.

* ./list\_comprehensions.py  
  A concise way to create lists.

***

### 📁 Data Transformations

* ./map\_vs\_replace.py  
  The difference between `map()` and `replace()` in pandas.

* ./missing\_values.py  
  How to handle missing values (`NaN`, `pd.NA`, `None`).

* ./restructuring.py  
  Examples of restructuring code for better readability and maintainability.

***

### 📁 Robust Code

* ./error\_handling.py  
  How to handle errors using `try/except`.
